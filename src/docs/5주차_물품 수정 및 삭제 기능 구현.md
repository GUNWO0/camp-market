- 사용자와 아이템 간 연관관계 유지
- 테스트용 유저를 자동으로 설정하여 기능 확인

---

## 🔧 구현 기능

### 1. 물품 수정 기능
- 기존 물품 정보를 `item-form.mustache`에 불러와서 수정 가능
- 수정 후 `itemRepository.save(item)`를 통해 DB에 반영
- 경로: `/items/{id}/edit`

### 2. 물품 삭제 기능
- 삭제 버튼 클릭 시 해당 아이템을 DB에서 제거
- 경로: `/items/{id}/delete`
